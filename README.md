### How to run typescript (lexer and parser) Example
If you have already set up `node` or `ts-node`, skip step 1 - 4
1. Install node.js following the instruction.
2. Clone this example repo to your local
3. Change directory to the repo where the `package.json` lives
4. `npm install`
5. Example runs: `npx ts-node 02_lexer/tiny_lexer.ts "let sum = a + 10;"`
